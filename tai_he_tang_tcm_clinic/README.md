# 泰和堂中医诊所网站

这是一个展示泰和堂中医诊所信息的网站项目。

## 项目结构

- `frontend/`: 前端静态文件
  - `index.html`: 主页
  - `herbs.html`: 中药材展示
  - `knowledge.html`: 中医知识
  - `virtual-tour.html`: 虚拟导览
  - `css/`: 样式文件
  - `js/`: JavaScript 文件
- `backend/`: 后端服务
  - `app.py`: Flask 应用
  - `requirements.txt`: Python 依赖

## 本地运行

1. 前端：
   - 直接打开 `frontend/index.html` 即可查看

2. 后端：
   ```bash
   cd backend
   pip install -r requirements.txt
   python app.py
   ```
